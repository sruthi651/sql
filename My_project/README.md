# 🚀 Introduction

This project analyzes thousands of real-world **Data Analyst job postings** using SQL to uncover valuable insights into the current job market.

The objective was to answer business questions regarding:

- 💰 Highest-paying Data Analyst jobs
- 🧠 Skills required for top-paying positions
- 📈 Most in-demand skills
- 💵 Highest-paying technical skills
- ⭐ Skills that provide the best combination of demand and salary

This project also served as an opportunity to strengthen my SQL skills by solving practical business problems using real-world data.

---

# 📖 Background

As the demand for Data Analysts continues to grow, understanding which skills employers value most has become increasingly important.

Using a dataset containing thousands of job postings, salaries, companies, and required skills, I performed SQL analysis to identify industry trends and answer meaningful business questions.

This project simulates the type of exploratory analysis that data analysts perform in real-world organizations.
